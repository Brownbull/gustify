# Story 4.4: Pantry Enhancements -- Manual Add, Category Filter, Unclassifiable Items

## Intent
**Epic Handle:** "Cook it, log it, grow from it"
**Story Handle:** "This story gives you full control of your pantry shelves -- add, filter, and flag what doesn't fit."

## Status: drafted

## User Story
As a user, I want to manually add ingredients to my pantry, filter by category, and flag items I can't classify, so that I have full control over my inventory.

## Acceptance Criteria

### AC-1: Manual Pantry Add
- **Given** the pantry view
- **When** I tap "Agregar ingrediente"
- **Then** a picker appears showing canonical ingredients (searchable, categorized) with quantity and unit fields

### AC-2: Manual Add Saves
- **Given** I select an ingredient, enter quantity and unit
- **When** I confirm
- **Then** the ingredient is added to my pantry with today as purchase date

### AC-3: Category Filter
- **Given** the pantry view
- **When** I select a category filter (Proteinas, Verduras, Frutas, Lacteos, Granos, Condimentos, etc.)
- **Then** only pantry ingredients in that category are shown

### AC-4: Show All Categories
- **Given** a category filter is active
- **When** I tap "Todas"
- **Then** all pantry ingredients are shown again

### AC-5: Unclassifiable Items
- **Given** the item mapping flow
- **When** I cannot find a matching canonical ingredient for a Gastify item
- **Then** I can tap "No clasificable" and the item is queued for review (stored in `unknownItems/`)

## Architectural ACs

- **AC-ARCH-LOC-1:** ManualPantryAdd component at `src/components/ManualPantryAdd.tsx`

## Dev Notes

- The `IngredientPicker` component already exists (`src/components/IngredientPicker.tsx`) and provides searchable ingredient selection with category badges. Reuse it for manual pantry adds.
- Category filter derives available categories from the canonical ingredients already loaded. The `IngredientCategory` type defines the categories.
- The unknown items service (`src/services/unknown-items.ts`) already exists. It may need a `submitUnknownItem()` method added.
- Manual pantry add uses the same `addToPantry()` service from `src/services/pantry.ts` -- no new Firestore logic needed.
- The category filter is client-side only (filter the already-loaded pantry items by their ingredient category).
