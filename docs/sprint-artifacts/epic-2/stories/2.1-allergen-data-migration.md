# Story 2.1: Allergen Data Migration for Canonical Ingredients

## Intent
**Epic Handle:** "No unsafe recipe reaches the screen"
**Story Handle:** "This story labels every ingredient in the pantry with its allergen warnings."

## Status: drafted

## User Story
As a developer, I want canonical ingredients to include allergen data, so that the system can filter recipes by dietary safety.

## Acceptance Criteria

### AC-1: Allergen Data on All Ingredients
- **Given** the canonical ingredients collection
- **When** allergen data migration is complete
- **Then** every ingredient has an `allergens` string array (e.g., `['gluten']`, `['dairy']`, or `[]` for none)

### AC-2: Standard Allergen Categories
- **Given** the allergen taxonomy
- **When** assigning allergens to ingredients
- **Then** values are drawn from a fixed set: `gluten`, `dairy`, `eggs`, `shellfish`, `nuts`, `soy`, `fish`

### AC-3: Idempotent Migration
- **Given** the migration script has already been run
- **When** executed again
- **Then** allergen data is updated (not duplicated) on existing ingredients

### AC-4: Type Safety
- **Given** the CanonicalIngredient TypeScript interface
- **When** updated with allergens field
- **Then** all code referencing ingredients compiles without errors

## Architectural ACs

- **AC-ARCH-LOC-1:** Allergen data added to `scripts/data/canonical-ingredients.ts`
- **AC-ARCH-PATTERN-1:** `allergens: string[]` field added to `CanonicalIngredient` interface in `src/types/ingredient.ts`
- **AC-ARCH-PATTERN-2:** Seed script updated to write allergen data alongside existing fields
- **AC-ARCH-NO-1:** No new collection — allergens are a field on the existing `canonicalIngredients/{id}` documents

## Dev Notes

- This is a data migration, not a UI change. No visual changes to the app.
- Common allergen mappings for Chilean ingredients:
  - Gluten: harina, pan, pasta, avena
  - Dairy: leche, queso, mantequilla, crema, yogurt
  - Eggs: huevo
  - Shellfish: camaron, cholga, almeja, jaiba
  - Nuts: mani, nuez, almendra
  - Soy: salsa de soja
  - Fish: salmon, merluza, atun, reineta, jurel
  - Most produce, meat, and spices: `[]` (no allergens)
- The existing seed script is idempotent (checks by ID before writing). The allergen field will be merged into existing documents.
